# GemmaTaiga: Offline AI Eco-Guardian for Khovsgol Wilderness

GemmaTaiga is an offline-first AI assistant designed for environmental protection and tourist safety in the Khovsgol region of Mongolia. 

## 🌲 The Problem
The most ecologically sensitive areas of Khovsgol have no network coverage, making cloud-based AI inaccessible for emergency and educational guidance.

## 🚀 Key Features
- **Offline Inference:** Runs locally on edge devices using `llama.cpp` and Gemma 4.
- **Eco-Education:** Provides guidance on sustainable camping and fire prevention.
- **Wildlife Awareness:** Explains the importance of local species and biodiversity.
- **Safety First:** Critical emergency protocols accessible without any internet signal.

## 🛠 Tech Stack
- **Model:** Google Gemma 4 (Quantized to 4-bit)
- **Engine:** llama-cpp-python
- **Framework:** Python / C# for the dashboard interface

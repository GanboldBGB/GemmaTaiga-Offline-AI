# GemmaTaiga: Offline Eco-Guardian for Khovsgol Wilderness

GemmaTaiga is an offline AI assistant designed for environmental protection and tourist safety in the Khovsgol region of Mongolia. 

## Key Features
- **Offline Inference:** Runs locally on edge devices using `llama.cpp`.
- **Eco-Education:** Provides guidance on sustainable camping and fire prevention.
- **Wildlife Awareness:** Uses Gemma 4 to explain the importance of local species.
- **Safety First:** Emergency protocols accessible without internet.

## Tech Stack
- **Model:** Google Gemma 4 (Quantized to 4-bit)
- **Engine:** llama-cpp-python
- **Deployment:** Edge-ready (Mobile/Laptop)
